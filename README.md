## Folder List
- images
  - A collection of image files (jpg, png, svg) used throughout the website
- scripts
  - Contains the script files for the website. Current contents:
    - skusuks.js (Contains logic for opening and closing the hamburger navigation menu, the product modal windows, including dynamically updating the modal windows with the current product, image and price, updating the shopping cart using local storage, button click and event handlers, phone number formatting,  
- styles
  - Contains multiple style sheets for the website, each serving a different purpose. File names include:
    - pages
        - cart.css (style rules for shopping cart and related elements)
        - contact.css (style rules for contact page)
        - shop.css (style rules for apparel and accessory pages)
    - main.css (imports of all other CSS files)
    - components.css (style rules for components like modal windows, forms, and buttons)
    - base.css (general style rules like fonts, colors, and background images)
    - layout.css (style rules for navigation, footer, and hero image)
    - skusuks_styles.css (original single CSS file containing all style rules)
- accessories.html
- apparel.html
- cart.html
- contact.html
- donate.html
- index.html
- preorder.html
- template.html
